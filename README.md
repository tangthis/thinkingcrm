Thinking CRM
------

**Thinking CRM** 是一款Java Web开发的后台管理系统。使用Spring + Hibernate + Jquery + Bootstrap等技术。


#### 基本功能：
> * 封装了通用的CRUD，减少重复代码
> * 基础的权限管理
> * Bootstrap简洁风格页面
